# Quantifying-wetland-carbon-emissions
This is a repo for Data+ and Climate+ project #21: Quantifying wetland carbon emissions in Southeastern US. 

## Files & Folders
- app.py: Shiny application, this is where users can interact with our UI to input data to produce data visualizations and predict carbon emissions in the form of NEE (net ecosystem exchange)
- requirements.txt: List of modules to install to run all code successfully
- util.py: Contains some utility functions for this project
- venv: Contains the virtual environment used in development
- data: Contains the datasets utilized
- src: Contains additional code
